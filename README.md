## 📌 About The Project

The **Bakery Website** is a modern, elegant, and fully responsive landing page designed for showcasing bakery products like breads, pastries, and custom cakes.

This project focuses on:

* Clean UI design
* Smooth user experience
* Mobile-first responsiveness
* Aesthetic visual hierarchy

---

## 🎯 Problem

Many bakery or small business websites:

* Lack modern UI
* Are not responsive
* Provide poor user experience
* Fail to attract customers visually

---

## 💡 Solution

This project solves these issues by implementing:

✨ Modern layout using **Flexbox & Grid**
✨ Soft pastel color palette for brand feel
✨ Interactive hover animations
✨ Clean typography and spacing
✨ Fully responsive design across devices

---

## 🛠️ Tech Stack

<p align="center">
  <img src="https://skillicons.dev/icons?i=html,css" />
</p>

* HTML5
* CSS3
* Google Fonts (Poppins)

---

## ✨ Key Features

✔️ Fully Responsive Design
✔️ Glassmorphism Header
✔️ Hero Section with Overlay
✔️ Product Feature Cards
✔️ CTA Buttons
✔️ Smooth Hover Effects
✔️ Clean & Structured Layout

---

## 🧠 What I Learned

* Responsive Web Design
* UI/UX Principles
* Layout structuring using Flexbox & Grid
* Visual hierarchy and spacing
* Writing clean and maintainable code
